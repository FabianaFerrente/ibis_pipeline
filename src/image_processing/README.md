# Image Processing Tools

High-level image operations for pre- and post-calibration processing, region masking, and grid interpolation.

## Modules

- `remove_residual_prefilter.py` – Spectral normalization step to remove prefilter residuals
- `feature_extraction.py` – Tools for feature identification (optional/placeholder)
- `aperture_masks.py` – Create and apply spatial masks or ROI selections
- `wavelength_grid.py` – Generate equidistant wavelength grids
