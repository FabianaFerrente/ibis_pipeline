# Utility Functions

General helper functions used throughout the pipeline. This module includes array manipulation, structure generation, input/output handling, and more.

## Modules

- `text_utils.py` – Text parsing and string splitting (e.g. get_words)
- `ibis_structures.py` – Create and populate IBIS data structures
- `user_io.py` – Pause, user prompts, cursor-based input (interactive)
- `array_utils.py` – Resizing, reshaping, padding arrays
- `image_displace.py` – Cross-correlation based image registration
- `norm_angle.py` – Normalize angles within a given interval
- `time_conversion` – Time conversion utilities

## Notes

These functions aim to simplify development and avoid redundancy across modules.
